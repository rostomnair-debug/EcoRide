# Charte graphique

## Palette
- Vert principal : #1A9A5F (boutons, accents)
- Vert clair : #6AD39C (badges, succès)
- Bleu-gris : #1E2A38 (header/footer texte)
- Gris clair : #F5F7FA (fonds cartes)
- Texte principal : #1F2933

## Typographie
- Titres : Poppins / Montserrat (sans-serif, bold) — choix pour leur lisibilité et un rendu moderne/éco, angles doux rappelant l’univers “nature”.
- Corps : Inter / system sans-serif — excellente lisibilité écran, neutralité qui laisse la place aux accents verts.

## Icônes
- Bootstrap Icons (leaf, coin, shield-check, chevrons, star).

## Style d’interface
- Cartes avec bordure légère et ombre douce.
- Badges éco (vert) et CO₂ (gris/vert clair), badges statut.
- Modales avec header coloré en vert pour confirmations.
- Pagination compacte avec boutons précédent/suivant.

## Maquettes (à fournir en PDF)
- Desktop : accueil, liste covoiturages, détail trajet.
- Mobile : accueil, liste covoiturages, espace profil.
- Suivre la palette + typographie ci-dessus, fonds clairs, accents verts, pictos nature (feuilles) et coin pour crédits.

## Justification des choix
- **Couleurs** : le vert principal (#1A9A5F) rappelle l’écologie et crée un call-to-action distinctif. Le vert clair (#6AD39C) adoucit les badges et retours succès. Le bleu-gris (#1E2A38) apporte contraste pour les textes structurants et ancre la crédibilité. Les gris clairs (#F5F7FA) évitent la fatigue visuelle sur les fonds.
- **Typo** : Poppins/Montserrat en titres pour un ton chaleureux et contemporain ; Inter en corps pour la lisibilité longue durée et un rendu neutre. Ce duo garde une hiérarchie claire sans surcharge.
- **Iconographie** : Bootstrap Icons pour cohérence et rapidité d’intégration ; feuilles et boucliers pour l’aspect éco/sécurité, pièce/coin pour le système de crédits.
